Extraction and recognition of pre-segmented pointers in medical images

this was my main project as a L3 student, it was a collaboration project with the South Dakota university and the NIH, our work was to extract and recognize connected components that resemble the given templates and either frame them or colorize them, we use both of Jaccard index and the neural networks to fulfill our task.

As a part of my study in the AI option I had to develop a connect 4 game that uses at least the min, max algorithm and the alpha, beta pruning, personally I wanted to add another algorithm because I found it amusement, so I have searched online for the algorithms working in the games and I used the MCTS algorithm

Media Library

This project is about creating a media library in java using sockets, threads, server programing, and email notifications.

The program administrates a library, the clients can borrow reserve or return books or DVDs, the DVDs can be locked by the age or open to all ages, if the client didn’t return the book or the DVD in three weeks then they will be banned for a month
